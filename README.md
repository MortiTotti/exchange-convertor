## Explanation
This is a simple react native app for converting the currencies to each other.

### Installation
- Run **`npm i`**.

### Run
- To run the application, just open a command prompt and type **`yarn start`**

### How to run the test
To run the test cases, in command prompt type: **`npm test`**
All test cases are in **`./src/__tests__`**


### Server API
The end point for this application is in **`http://data.fixer.io/api/`**
To use the api you should provide a api key which you can find it in **`./src/constants/YAS`**.
